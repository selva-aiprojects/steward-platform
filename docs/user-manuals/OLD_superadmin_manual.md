# StockSteward AI: Superadmin Command Manual
## Role: System Administrator

Welcome to the **Command Center**. As a Superadmin, you hold the highest level of authorization within the StockSteward AI ecosystem.

### 1. System Oversight
- **Global Strategy Monitoring**: View all active AI strategies and their resource consumption.
- **Node Telemetry**: Monitor the health of execution nodes and websocket tunnels.
- **Audit Logs**: Access the immutable global audit trail for all system events.

### 2. User & Access Management
- **Role Assignment**: Elevate or restrict user permissions (Trader, Auditor, Business Owner).
- **Session Security**: Force logout or terminate suspicious agent activities.
- **Credential Rotation**: Standard security procedures for API and database access.

### 3. Infrastructure Config
- **API Gateways**: Configure endpoints for brokerage connections and market data providers.
- **Model Tuning**: Adjust the AI's risk-reward bias for autonomous execution.

> [!IMPORTANT]
> All Superadmin actions are logged cryptographically. Ensure your session is secured with 2FA where available.

## Create Users (RBAC)
1. Log in as Superadmin.
2. Go to **User Mgmt**.
3. Click **Add User** and choose role: SUPERADMIN / BUSINESS_OWNER / TRADER / AUDITOR.
4. Set password, trading mode, and risk tolerance.
5. Save and share credentials securely.
